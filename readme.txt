# ai智能生成简历

## 项目概述

这个网页集成了讯飞星火认知大模型的强大功能。该项目提供了一个直观、美观的用户界面，使您能够与先进的 AI 大模型进行流畅对话，获取信息，并利用 AI 的理解和生成能力解决各种任务。

## 核心功能

- **实时对话**：与讯飞星火大模型进行流畅、自然的实时对话交互
- **对话历史**：自动保存和管理您的聊天历史，方便继续之前的对话
- **响应式设计**：在不同设备上提供一致且优化的用户体验

## 技术栈

- **前端框架**：Vanilla JavaScript + Vite 构建系统
- **API 集成**：WebSocket 实时通信
- **加密安全**：使用 crypto-js 实现安全的 API 签名验证
- **样式**：现代化的 CSS3 设计和动画效果
- **存储**：LocalStorage 对话历史持久化

## 快速开始

### 前提条件

- Node.js (版本 14.0 或更高)
- 讯飞开放平台 API 访问凭证

### 安装

1. 下载此项目并且解压，拖入vscode

2. 安装依赖

在代码根目录下的终端输入npm install

3. 配置
编辑 .env 文件，填入您的 API 凭证（代码内的各id是我本人的）
# VITE_APP_SPARK_APPID='您的APPID'
# VITE_APP_SPARK_APISECRET='您的APISECRET'
# VITE_APP_SPARK_APIKEY='您的APIKEY'


### 启动开发服务器

在代码根目录下的终端输入：npm run dev

服务拉起后，访问拉起的网址即可开始与 AI 对话。

## 使用指南

1. 在输入框中键入您的问题或指令
2. 点击"发送"按钮或按 Enter 键提交
3. 实时查看 AI 的回复内容
4. 对话完成后，可以继续追问或开始新话题
5. 使用"清除历史聊天记录"按钮可以开始全新的对话
